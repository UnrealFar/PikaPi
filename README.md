# PikaPi

> This is the official repository for the PikaPi Discord bot.

PikaPi is a Discord based on Pokemon.

## Run the bot yourself
> It is highly unrecommended to host the bot yourself. You will need the metadata cluster(which is private) in your database. You may not ask me for this data(I will not give it) no matter what the circumstances are.

Now, if you still want to do it then follow the given steps.
To run the bot yourself, create a .env file and fill in this information:

```py
token = "<YOUR-BOT-TOKEN>"
mongo_uri = "<URI-OF-YOUR-MONGODB-CLUSTER>"
```

Now run "sh main.sh"
