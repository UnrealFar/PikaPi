# PikaPi
AS OF NOW, THIS BOT IS STILL BEING DEVELOPED AND IS IN ITS ALPHA PROGRAM!
-A fun pokemon-realted bot in Discord!
-With this bot, you can catch pokémon in the wild, level your pokémon up, have battles with your friends and other people, trade, view the pokedex info of all the pokemon, and almost everything you can imagine in the world of pokemon!
-Invite this bot to your server with the p!invite command!
