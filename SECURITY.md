## Reporting Security Issues

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them to ModMail in the [Discord server](https://top.gg/servers/871048037768790016).

You should receive a response within 24 hours. If for some reason you do not, please follow up via DM to ensure we received your original message.

Please include the information below (to help us understand the issue better):

- Issue Type
- Full paths to the Source File(s)
- Location of Source Code (tag/branch/commit)
- Any configuration mandatory to reproduce the issue
- Step-by-step instructions to reproduce the issue
- Proof (screenshot, exploit code)
- Issue Impact and how it could be used maliciously.

## Preferred Languages

We prefer all communications to be in English.
